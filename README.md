# technical-challenge-be

- Aplicación basada en Spring Boot para guardar información de usuarios y datos telefonicos

# Stack Técnologico

* Spring boot 2.2.6.RELEASE
* Java 8
* Bean Validation
* Swagger
* Spring JPA Starter
* H2 Database
* GCP Pub/Sub 
* GCP Cloud Functions 
* Mailtrap
